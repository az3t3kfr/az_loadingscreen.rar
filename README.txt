🚀 1. INSTALLATION

1. Téléchargez le dossier "md_loading".
2. Glissez-le dans votre dossier "resources" de votre serveur FiveM.
3. Allez dans votre fichier "server.cfg".
4. Ajoutez la ligne suivante : ensure md_loading
5. Redémarrez votre serveur.

⚠️ IMPORTANT : Assurez-vous qu'aucun autre écran de chargement n'est activé 
simultanément dans votre server.cfg pour éviter les conflits.

---

⚙️ 2. CONFIGURATION GLOBALE (Fichier : config.js)

Toute la magie de MD STUDIO se trouve dans le fichier 'config.js'. 
Ouvrez-le avec un éditeur de texte (Notepad++, VS Code) pour modifier :

• Nom du Serveur : Changez "MD STUDIO" par le nom de votre projet.
• Couleurs : Modifiez "AccentColor" (code Hexadécimal) pour changer la couleur 
  de la barre de chargement, des boutons et des titres.
• Staff : Ajoutez ou supprimez des membres en respectant le format. 
  Vous pouvez définir le nom, le rôle, la photo et la couleur de chaque membre.
• Règles : Modifiez la liste pour qu'elle corresponde à votre règlement.
• Touches : Paramétrez les touches (F2, F3, etc.) pour aider vos nouveaux joueurs.
• Site Web : Remplacez le lien par votre boutique ou votre Discord.

---

🖼️ 3. GESTION DES MÉDIAS (Dossier : assets)

Pour que l'aspect visuel de MD STUDIO soit unique, placez vos fichiers dans 
le dossier 'assets' en respectant les noms suivants (ou changez-les dans config.js) :

• Fond d'écran : assets/fond.jpg (Format recommandé : 1920x1080)
• Logo : assets/logo.png (Format recommandé : PNG Transparent)
• Musique : assets/music.mp3 (Format recommandé : .mp3 léger)
• Photos Staff : Placez les visages de votre équipe ici (Ex: assets/LOGO2.png).

---

🛠️ 4. PERSONNALISATION AVANCÉE

• CSS (style.css) : Si vous avez des connaissances en design, vous pouvez 
  modifier les arrondis, les flous (backdrop-filter) et les animations.
• HTML (index.html) : Vous pouvez modifier l'ordre des onglets ou ajouter 
  des descriptions personnalisées dans les sections.
